## Список экранов приложения

| № | URL | Название | Доступ |
|---|-----|----------|--------|
| 1 | `/login` | Вход в приложение | Все |
| 2 | `/reset-password` | Сброс пароля | Все |
| 3 | `/index` | Главный экран | Авторизованные |
| 4 | `/knowledge` | База знаний | Авторизованные |
| 5 | `/contact` | Форма ОС / Запись на консультацию | Авторизованные |
| 6 | `/route` | Экран маршрута (шаги А-1, А-2, Б-1, Б-2) | Авторизованные |
| 7 | `/result` | Экран результата (Документ 1 + Документ 2) | Авторизованные |
| 8 | `/admin` | Административная панель | Только `admin` |

### Экран маршрута `/route`

Один переиспользуемый экран для всех четырёх маршрутов.
Тип маршрута передаётся как параметр: `/route?type=A1`, `/route?type=A2`, `/route?type=B1`, `/route?type=B2`.

Элементы, присутствующие на всех шагах маршрута:

- Хлебные крошки: `Главный экран → [тип работы] → [маршрут]`
- Индикатор прогресса (шаг N из N)
- Кнопка «Назад» — предыдущий шаг, ответы сохранены
- Кнопка «Начать заново» — сброс, возврат на `/index`

### Экран результата `/result`

Также переиспользуемый — отображает итог нужного маршрута.

Элементы:

- Блок Документ 1: «Обращение в юридическую службу» + кнопка скачать PDF
- Блок Документ 2: «Инструкция-чеклист» + кнопка скачать PDF
- ⚠️ Для Б-2: предельная дата направления ответа (выделена отдельно)
- Кнопка «Начать заново» → `/index`
